# BTUH-tweet-crawler

## Introduction

* BTUH(BetterThanYourHands)-tweet-crawler 
* 트위터 api는 비싸고!
* Http 따보려니 암호화 되있고!
* 직접 손으로 긁는게 나을 것 같아 만들었습니다.



>  손으로 긁는 것 보단 낫습니다..



#### Installation

- [Chrome Web Driver](http://chromedriver.chromium.org/downloads)



#### Run

- 현재 날자 ~ {%Y-%m-%d-%H:%M:%S}  까지 데이터를 크롤링합니다.

```
python3 tweetCrawler.py {keyword} {%Y-%m-%d-%H:%M:%S}
```
